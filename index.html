Password Manager

I created this password manager to be a secure and reliable solution to manage my login credentials. Unlike commercial password managers that store passwords in the cloud, this password manager generates random passwords and saves them locally on my machine, ensuring that passwords are not vulnerable to online security breaches.

Feel free to fork the code on my Github: Password Manager v2.
